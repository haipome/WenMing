## 简介

自由不是人类的天然选择，而是特定条件下的偶然产物。

本书追踪了两千五百年的政治实验——从希腊城邦、罗马共和、到荷兰、英国、美国——提炼出一个核心发现：自由社会不是因为某个思想家的启蒙而诞生的，而是因为一笔交易。当中产阶级足够强大，统治者不得不用权利交换他们的税收、劳动和忠诚，自由就作为这笔交易的副产品出现了。没有一个案例是因为有人相信"自由是好的"而主动选择了自由。

但这笔交易正在瓦解。

书中用大量篇幅分析了自由秩序正在遭受的三重打击：制度从内部朽烂——制衡机制被一道一道拆除，每一次拆除都有真实的缺陷作为理由，不可反驳的道德叙事作为正当性，合法的程序作为手段，不可逆的后果作为结局；中产阶级在经济和人口上同时萎缩，自由社会的社会基础正在消失；而AI正在把这一切加速到一个全新的量级。

AI是本书最尖锐的部分。作者的论证不是"AI会取代工作"这种泛泛之谈，而是一个精确的政治经济学推演：AI不是从底部替代体力劳动，而是从中间塌陷——打击的恰恰是律师、会计、程序员、分析师这些构成中产阶级核心的白领认知岗位。这不是在削弱中产阶级的边缘，而是在挖空它的中心。与工业革命不同，AI把认知劳动的边际成本压到接近零，上升通道在收窄而不是扩展，连向下坠落的退路都在被封堵。当中产阶级失去经济议价权，维系自由社会的那笔交易就失去了基础。同时，AI让信息控制、社会监控和暴力执行的成本趋近于零，天平两端同时倾斜——中产阶级的筹码在减少，统治者的工具在增强。

作者将此与罗马共和国的衰亡做了精确的类比。AI就是我们这个时代的马略改革——它解决了一个真实的效率问题，但副作用是切断了"经济参与→纳税→议价权"这条维系自由社会的生命线。罗马从自耕农破产到凯撒渡过卢比孔河，中间大约一百年。制宪者设计的美国宪政机器，可能在AI的冲击下以更快的速度走完同样的路。终局是形式上的共和，实质上的帝制——选举还在，议会还在，宪法一个字都没改，但它们已经不能保护任何人的自由了。

面对这个前景，书的最后两章提出了建设性的回应：一套基于两千五百年政治实验教训的稳定共和制度设计——议会制而非总统制、虚位总统、不对等两院、真正的联邦制、独立的货币权——以及对区块链作为去中心化自由实验的深入分析。作者并不乐观，但他指出：自由倒过很多次，只要种子库还在，竞争还在，它就会在某个意想不到的角落重新长出来。
